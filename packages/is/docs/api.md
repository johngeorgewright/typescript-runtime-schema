## API
### is
Default export.

#### Signature
`function is<T>(value: any) => value is T;`

#### Type Parameters
Name | Constraint | Default value | Description |
------ | ------ | ------ | ------ |
`T` | N/A | N/A | The type we are type guarding against. Must be a serializeable value |

#### Parameters
Name | Constraint | Default value | Description |
------ | ------ | ------ | ------ |
`value` | N/A | N/A | The value we are type guarding

#### Returns
boolean: `value` is `T`
